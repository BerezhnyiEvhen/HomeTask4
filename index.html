git add .- додавання(оновлення) усіх файлів в репозиторії в обраній гілці; git commit -m "текст коміту" - додавання коміту; git checkout "name of the branch" - переключення з гілку на обрану гілку; git branch -v - термінал покаже список із всіх гілок репозиторію; git push origin "name of the branch" - встановлення зв'язку із відаленним репозиторієм та оновлення до останньої версії - останніх змін;
